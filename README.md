# AI-Based Market Trend Analysis on Sports Products

## Project Overview
This project focuses on analyzing market trends for sports products using AI-based techniques.
The objective is to understand demand patterns, forecast future sales, and analyze customer sentiment to support data-driven business decisions.

## Problem Statement
Sports product businesses often face challenges in understanding demand trends and predicting future sales.
Manual analysis is time-consuming and may not provide accurate insights.

## Approach
The project follows a structured data analytics and AI approach:
- Data preprocessing and cleaning
- Demand trend analysis using KMeans clustering
- Sales forecasting using time-series analysis (Exponential Smoothing)
- Sentiment analysis on customer reviews using NLP techniques

## Tools & Technologies
- Python
- Jupyter Notebook (Google Colab)
- Pandas, Matplotlib
- Scikit-learn
- Statsmodels
- TextBlob

## Key Results
- Products were grouped into different demand categories using clustering
- Future sales trends were forecasted for multiple sports products
- Customer sentiment was classified into positive, negative, and neutral categories

## Learnings
- Practical understanding of AI techniques applied to business problems
- Experience with machine learning, time-series forecasting, and NLP
- Improved skills in data analysis and visualization

## Conclusion
The project demonstrates how AI techniques can be applied to analyze market trends and support better business decision-making in the sports products domain.
